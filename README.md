# 🔥 Advanced Calculator System

A production-ready, full-stack web-based calculator system crafted with a premium UI, robust user authentication, dynamic history tracking, and usage analytics.

## ✨ Features

- **Advanced Mathematics**: Evaluates basic arithmetic, parentheses, and includes a scientific mode (`sin`, `cos`, `tan`, `log`, `sqrt`, `%`, `^`).
- **Secure Expression Parsing**: Mathematical strings are processed using a custom Python `ast` syntax tree evaluator to guarantee protection against arbitrary code execution (no `eval()` vulnerabilities).
- **User Authentication**: Secure JWT-based stateless authentication with `bcrypt` password hashing. 
- **Analytics Dashboard**: Tracks the total volume of calculations over time and your most-used operators, visualized beautifully with `Chart.js`.
- **History Management**: Automatically saves your calculations to PostgreSQL. Built-in support for UI pagination and exporting data to format `.csv`. 
- **Premium Aesthetics**: Engineered with deep custom CSS properties simulating a modern "glassmorphism" look natively adapting to Light & Dark themes.

## 🚀 Local Deployment Setup

### 1. Database Configuration (PostgreSQL)
Ensure you have PostgreSQL installed. Open pgAdmin4 or the `psql` console and create a new database.
```sql
CREATE DATABASE calculator_db;
```
*(By default, the app expects credentials `postgres:shakibalarman@localhost/calculator_db`. You can modify your connection string directly in `backend/app/database.py`).*

### 2. Backend Initialization
Open your terminal and navigate to the `backend` directory:
```bash
cd backend

# Create and activate a Python virtual environment
python -m venv venv
.\venv\Scripts\activate   # On Windows (If blocked, use the python path below directly)
# source venv/bin/activate  # On Mac/Linux

# Install all required backend dependencies
pip install -r requirements.txt

# Boot up the FastAPI Application
.\venv\Scripts\python -m uvicorn app.main:app --reload
```
You should now have the API running locally on `http://127.0.0.1:8000`. You can test endpoints safely via the autogenerated Swagger API page at `http://127.0.0.1:8000/docs`.

### 3. Frontend Initialization
Open up the frontend folder in any lightweight development server to avoid CORS constraints caused by standard `file://` protocols. 
```bash
cd frontend
python -m http.server 3000
```
Then simply navigate to `http://localhost:3000/login.html` in your web browser to enjoy!
